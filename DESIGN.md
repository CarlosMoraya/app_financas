
# Design System

## Paleta de Cores

| Token        | Cor     | Descrição         |
|--------------|---------|-------------------|
| `primary`    | #2563eb | Acento principal para botões |
| `secondary`  | #6b7280 | Tons neutros para textos |
| `background` | #f9fafb | Cor de fundo |
| `surface`    | #ffffff | Cor dos cartões |
| `error`      | #dc2626 | Mensagens de erro |
| `warning`    | #f59e0b | Mensagens de alerta |
| `success`    | #16a34a | Mensagens de sucesso |

## Tipografia

- Fonte: Inter
- Tamanhos: sm, base, lg, xl
